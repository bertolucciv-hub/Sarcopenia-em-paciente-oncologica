# Caso Clínico — Sarcopenia em paciente oncológica

Pacote final estático para publicação no GitHub e na Vercel.

## Estrutura
- `index.html` — página principal
- `assets/figura_1_cancer_gastrico_sarcopenia_funcional.png`
- `assets/figura_2_prescricao_treinamento_12semanas.png`
- `vercel.json`

## Publicação no GitHub
1. Crie um repositório novo.
2. Envie todos os arquivos deste pacote para a raiz do repositório.
3. Opcionalmente ative GitHub Pages.

## Publicação na Vercel
1. Importe o repositório no painel da Vercel.
2. Framework preset: `Other` ou `Static`.
3. Build command: deixe em branco.
4. Output directory: deixe em branco.
5. Deploy.

Como se trata de uma página estática, a Vercel publicará diretamente o `index.html`.
